# JustUI

> made with gulp / sass / typescript.

## Installing

Clone Repo

```
npm install

// Start Gulp Watch mode
npm run dev

// Build Production Mode
npm run build
```

## Build with

-   [gulp](https://gulpjs.com)
-   [SASS](https://sass-lang.com)
-   [Typescript](https://typescriptlang.org)
